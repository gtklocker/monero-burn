Monero [Proof-of-Burn](https://eprint.iacr.org/2019/1096.pdf)
===

Library that generates and verifies proof-of-burn addresses for any tag.

## Usage

The library can be built and tested using cargo:

```
cargo build
```

Run the CLI tool with:

```
cargo run -- cafecafe
```
